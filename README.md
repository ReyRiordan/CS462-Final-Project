# CS462-Final-Project

CS462 Deep Learning final project w/ Qigui Weng :^)


[dataset/](dataset/): De-identified RWJMS student clinical note evaluation dataset and benchmark, includes faculty evaluations, consensus evaluation, student-written notes, and master rubric.

[evaluate.py](evaluate.py): All code to evaluate an LLM of choice on our benchmark with any of the three confidence estimation methods.

[prompts/](prompts/): All prompts used for LLM evaluation.

[results/](results/): All result data obtained during study, also the output location when running evaluation code.

[analyze.py](analyze.py): All code to analyze results data according to performance metrics + data visualizations.

Proposal and Final Report are also included as PDFs.
